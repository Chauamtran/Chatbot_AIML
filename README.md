# Introduction
This is an example about building a chatbot using Artificial Intelligence Markup Language (AIML)
The example is running by a lightweight web framework **flask** 
# Requirements
All required packages are defined in **requirements.txt**, please install them first to start project

**Recommendation:**
You should install pyenv first to manage python versions (recommend 3.6), pipenv to create a virtual environment to the project  
# Information:
* **brains** folder to store models of chatbot
* **conversations** folder to store sample sentences that are pre-processed
* **data** folder to store knowledge base of chatbots
* **source/main** folder to store source code of chatbot
* **web** folder to create a simple UI to interact with chatbot
